特点

UIKit为您提供了全面的HTML、CSS及JS组件，
它们使用简单，容易定制和扩展。

LESS

UIkit 基于 LESS 开发，代码结构清晰简单，易于扩展和维护。

组件

体积小、反应灵敏的响应式组件，使用一致的、无冲突的命名规则。

定制

可根据 UIKit 基本的风格及样式，轻松地自定义创建出自己喜欢的主题样式。

响应式

根据不同的屏幕分辨率与上网设备，UIKit 会自动做出响应，提供一致的体验。

开放源码

UIKit 是一款开源的前端UI界面框架，
你可以无任何限制的使用 UIKit 来创建自己的风格。

MIT 许可协议

UIkit 为开源项目，基于 MIT 许可协议。它是完全免费的，你可以无任何限制的使用、复制、合并、发布及分发这个框架。

我们的英雄

非常感谢 LESS、jQuery、normalize.css及FontAwesome，没有这些强大的开源项目就不可能有UIkit。

积极的开发

UIkit 由 YOOtheme 团队建造，YOOtheme 通过丰富的经验创建着 Joomla 和 WordPress 主题，并支持着 Warp theme framework 项目。

加入我们

UIkit 是托管在 GitHub 上的开放项目，所有人都可以为它作出贡献。
欢迎给予我们一些反馈意见，并加入到开发中来！

贡献

你想要帮助我们，并参与一些开发工作或者编写文档？你只需在 GitHub “Fork” UIkit 并向我们发送一条pull请求即可。

发现BUG？

在 GitHub 上面打开一个 New Issue。请首先搜索已存在的 Issues，并确保包含所有相关的信息。

分享主题

你创建了一个漂亮的 UIKit 主题？我们非常欢迎把它分享给所有人，并把它添加到 UIKit 的定制器中，向我们发送一条 pull 请求吧。
